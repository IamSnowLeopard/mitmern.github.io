## Project Name:

## Description:

- Add to do items
- Delete items
- Cross out completed items

## How to Use:

1. Clone the repository or download the source code
2. Open TodoApp.html in a web browser to run the application
3. Type the to do item
4. Click "Done!" or "Delete" to modify the items
